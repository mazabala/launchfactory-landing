From Zero to Live:
Vibe Code Your First Online Product in a Weekend

A beginner's guide to building and launching a Hook Generator tool using AI

You do not need to know how to code. You need a computer, an internet connection, and the willingness to copy some commands.

This guide walks you through building the Hook Generator — a web tool where someone enters a topic and gets back 10 social media hook ideas based on proven templates.

By Sunday night, you'll have a live product deployed at your own domain name.

What the tool does

  • User enters a topic (e.g., "fitness coaching")
  • User picks a platform (Instagram, TikTok, Twitter, LinkedIn)
  • Clicks "Generate"
  • Gets 10 hook options to copy
  • Optionally submits their email via embedded form

How it actually works: The tool combines your topic with a JavaScript library of 100+ proven hook templates. Results are shuffled so each generation feels fresh. It is template-based, not AI. You are building a tool that mixes and matches templates, not a magic generator.

Who this is for

  • Anyone who posts on social media
  • Content creators who hate writing headlines
  • Marketers who need hook ideas fast

The Stack

  • Antigravity — Browser IDE with Claude Code
  • GitHub — Code storage
  • Claude Code — The AI that writes the code
  • Render — Where the product goes live
  • Tally — Email capture (embedded form, no backend)
  • Namecheap — Your domain name

Total cost: Free if you use the Render URL, around $10/year if you buy a custom domain.

Chapter 1: Get Your Tools Ready

What You're Doing

Setting up accounts. Takes about 20 minutes.

Step 1.1: Create a GitHub Account

  1. Go to github.com
  2. Click "Sign Up"
  3. Pick a professional username
  4. Enter your email and create a password
  5. Verify your account

Step 1.2: Open Antigravity

  1. Go to Antigravity and create an account
  2. Log in
  3. You should see a code editor interface

Leave this tab open.

Step 1.3: Connect GitHub to Antigravity

  1. In Antigravity, look for a GitHub icon or settings
  2. Click "Connect GitHub"
  3. Authorize the connection
  4. You should see your GitHub username appear

This way your code is saved and you do not lose anything.

Verify Before Moving On

  • You can log into GitHub
  • You can log into Antigravity
  • Antigravity shows your GitHub account connected

Chapter 2: Build the Hook Generator

What You're Doing

Using Claude Code to build the entire tool. This is the main work. Takes 2-4 hours.

Step 2.1: Open a New Project in Antigravity

  1. In Antigravity, create a new project or open from GitHub
  2. Name it: hook-generator
  3. Make sure it is empty to start

Step 2.2: Ask Claude Code to Build the Landing Page

Find the Claude Code chat in Antigravity. Paste this prompt:

Build a Hook Generator web tool. Single HTML page with:
- Dark theme, clean modern design
- Headline: "Turn any topic into 10 hook ideas"
- Subheadline: "Enter your topic, pick a platform, get hooks you can copy right now"
- Topic input field (text input with placeholder "e.g., fitness coaching")
- Platform selector: Instagram, TikTok, Twitter, LinkedIn (button group or dropdown)
- "Generate Hooks" button (prominent, full width on mobile)
- Results section: empty to start, shows 10 hook cards after generation
- Each hook card shows the hook text + a "Copy" button
- Mobile responsive with smooth animations

Claude will create files. Say yes to creating them. Preview the page.

Step 2.3: Add the Hook Template Library

Go back to Claude Code. Paste this prompt:

Add a JavaScript hook library to this page with at least 100 hook templates.

Organize them by type:
- Numbers: "The 1 thing...", "3 reasons why...", "5 ways to...", "The only X you need..."
- Questions: "Why does everyone...", "What if I told you...", "How come...", "Did you know..."
- Contrast: "Most people X, but...", "You think X but actually Y", "Unlike most people who..."
- Bold claims: "The truth about...", "Nobody tells you...", "Stop doing X...", "The secret to..."
- Curiosity: "You will never guess...", "This is why...", "Here's the secret...", "The real reason..."

Create a generateHooks function that:
- Takes the user topic (from the input field)
- Takes the selected platform
- Returns 10 hook options by combining the topic with template patterns
- Shuffles the results so they are different each time
- Shows a brief loading state (1 second) before showing results
- Adds the platform name to each hook naturally

This is a template-based generator, not AI. The function mixes your topic with existing templates.

Preview. Type a topic, pick a platform, click Generate. Does it work?

Step 2.4: Make the Copy Button Work

Go back to Claude Code. Paste this prompt:

Add clipboard copy functionality to each hook card.

When the user clicks "Copy" on any hook:
- Copy the hook text to clipboard
- Change the button text to "Copied!"
- After 2 seconds, change it back to "Copy"
- Make sure it works on mobile and desktop

Test this. Click a Copy button. Does it say "Copied!" and then change back?

Step 2.5: Create Your Tally Form

Tally lets you collect emails without any backend code.

  1. Go to tally.so and create a free account
  2. Create a new form with:
    • A short headline (e.g., "Save these hooks")
    • One email field (required)
  3. In the form settings, get the embed code
  4. Copy the iframe src URL

Your embed URL looks like:

https://tally.so/embed/YOUR-FORM-ID?alignLeft=1&hideTitle=1&dynamicHeight=1

Replace YOUR-FORM-ID with your actual form ID from Tally.

Step 2.6: Embed Tally Form Below Results

Go to Claude Code. Paste this prompt:

Add a Tally email form below the hook results section.

Use this iframe embed (replace YOUR-FORM-ID with your actual Tally form ID):
https://tally.so/embed/YOUR-FORM-ID?alignLeft=1&hideTitle=1&dynamicHeight=1

Style the iframe to be full width and have a minimum height of around 150px.
Add a small headline above the form: "Want more hooks? Drop your email."

Replace YOUR-FORM-ID with your actual Tally form ID.

Step 2.7: Test Everything

Preview the page:

  1. Enter a topic
  2. Pick a platform
  3. Click Generate
  4. See 10 hooks appear
  5. Click Copy on one hook
  6. Scroll down and confirm the Tally form is there
  7. Submit a test entry to your Tally form

If something breaks, tell Claude Code what went wrong and ask it to fix it.

Chapter 3: Deploy to Render

What You're Doing

Putting your product on the internet.

Step 3.1: Push to GitHub

Go to Claude Code. Paste this prompt:

Initialize git in this project if not already initialized.
Create a .gitignore file (ignore node_modules if any).
Commit everything with message "initial commit"
Push to GitHub.

Verify on GitHub that your files are there.

Step 3.2: Create a Render Account

  1. Go to render.com
  2. Click "Get Started"
  3. Sign up with your GitHub account
  4. Verify your email

Step 3.3: Deploy

  1. In Render, click "New" and select "Web Service"
  2. Connect your GitHub account
  3. Find and select your hook-generator repository
  4. Configure these settings:
SettingValue
Namehook-generator (or your choice)
RegionClosest to you
Branchmain
Root DirectoryLeave empty
RuntimeStatic
Build CommandLeave empty
Publish Directory./
  1. Click "Create Web Service"
  2. Wait 30-60 seconds for it to deploy
  3. You will see "Live" when it is done

Click the URL Render gives you. Your product is now live on the internet.

Save that URL. You need it for the next step.
